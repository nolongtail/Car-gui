# Car-gui

## Byte value meaning
- 1st hex is defining state, 2nd hex is defining timeleft
### Traffic Light State representative in HEX format
| hex | state   |
| --- | :----   |
| 1   | red     |
| 2   | green   |
| 3   | yellow  |
| f   | standby |
### timeleft
the hex value is equal to the time left in second
